derivation_response(env, **options)

Calls the derivation endpoint passing the request information, and returns the Rack response triple.

It uses a trick where it removes the derivation path prefix from the path info before calling the Rack app, which is what web framework routers do before theyâ€™re calling a mounted Rack app.
